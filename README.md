# My little Twitter Bot build with Tweepy 3.10.0 in Python 3.7.3

(Debian Kernel Shell)

$ mkdir my_botaki

$ cd my_botaki

$ python3 -m venv venv

$ source ./venv/bin/activate

$ pip install tweepy

$ pip freeze > requirements.txt

A developer account on twitter required.
